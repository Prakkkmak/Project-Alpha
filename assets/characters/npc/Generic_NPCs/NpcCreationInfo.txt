Each folder contains all the layers you need to make the NPC animations indicated by the folder name.

To Create generic NPCs, choose a character layer and then combine one layer from each of the outfit groups below.
You can leave any group empty if you wish. For example, not adding gloves to your NPC.



+ Outfit base layers:

	- Shirt
	- Doublet


+ Chest layers:

	- Blouse
	- Jacket
	- Shoulder pads
	- Toga
